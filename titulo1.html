MATERIALES

Cerámica y Alfarería: Arcilla para modelado a mano. Engobes y esmaltes para decoración. Madera para la creación de esculturas y objetos decorativos. Pinturas para la decoración de las piezas talladas. Trabajo con Metales: Hierro y otros metales para forja. Técnicas de cincelado para agregar detalles. Trabajo con Cuero: Cuero para la confección de accesorios. Tinturas para teñir el cuero y resaltar detalles. Trabajo con Piedra: Piedra para la escultura y elementos decorativos. Pulido y acabado para realzar la belleza de las piezas. Textiles y Bordados: Técnicas de teñido para colorear hilos y telas. Bordados a mano para detalles intrincados. Confección de Joyería: Metales preciosos para la creación de joyas. Filigrana y técnicas de incrustación para detalles decorativos.
